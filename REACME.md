# Yarn Start Project 
 

install dep
```
yarn
```
start
```
yarn dev
```
build
```
yarn build
```
preview or serve
```
yarn preview
```



# Npm Start Project 
 

install dep
```
npm i
```
start
```
npm run  dev
```
build
```
npm run  build
```
preview or serve
```
npm run preview
```
